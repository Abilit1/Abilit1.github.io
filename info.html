Information

Bnax is a project still in beta versions that will allow you to create a personalized discord bot, without the need for effort. This project will only have 3 plans at the moment in which one is in development, the bost do not have a host so the client must host it or the bot will simply be constantly monitored, the problem with this is that it can have frequent crashes. It is necessary to be on the discord server, all the bots created, tests and errors are recorded there.

How tu use?

To make the bot go to the discord developer portal page and create an application in the applications section, after that go to the "bot" section and press add bot consecutively copy the token of the bot and put it in the form that appears when claiming one of the plans.
